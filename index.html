Forms

Forms is a tool that lets you design online forms.
With it you can easily convert your forms and make them available online without needing to know how to write any code.

Form Builder

The form builder lets you drag and drop various field elements onto a form. Use it to design and build online forms for data collection. 

Learn More

Approval Builder

The approval builder lets you design forms that require some sort of approval. You can customise the approval process however you want. When a user submits a form, the approvers will be notified and prompted to login and approve/deny the submitted entry. 

Learn More

Logic Builder

The Logic Builder lets you build logic into the form without coding. This allows you to design smart forms that perform tasks like to show/hide fields, send different emails, communicate with other applications etc.

Learn More

Entries Manager

The entries manager lets you collect, store and manage data that a form has captured. You can filter, edit, delete, approve and export entries using the form manager.
